### Mina svar


**Vilka ekvivalensklasser har uttrycken?**

1a. x > 100

Svar: 3st. Under 100 / 100 / Över hundra.

1b. y == 42

Svar: 3st. Under 42 / 42 / Över 42.

1c. len(text) >= 5

Svar: 3st. Under 5 / 5 / Över 5. 

1d. z == True

Svar: 2 st. En true (z) / en false.

1e. 8 < v < 16

Svar: 5st. 1 under 8 / 8 / v / 16 / 1 över 16.

1f. w == 32 or w == 64 or w == 128

Svar: Minst 4. 32, 64, 128 och minst en siffra utöver det.

1g. if x < 5: … elif x < 10: … elif x < 15: … else … 

Svar: 4st. 1 mindre än 5 / 1 mellan 5 - 9 / 1 mellan 10 - 14 / 1 över 15.
